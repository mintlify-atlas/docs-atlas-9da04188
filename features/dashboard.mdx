## Overview

The dashboard provides comprehensive analytics for community financial oversight, including budget execution metrics, rendition status tracking, and automated alerts for pending actions.

## KPI Endpoint

The main dashboard data is served from a single endpoint:

```python
@api_view(['GET'])
@permission_classes([IsAuthenticated])
def dashboard_kpis(request):
    """Returns general KPIs for the dashboard filtered by community."""
    user = request.user
    if not hasattr(user, 'comunidad') or not user.comunidad:
        return Response({"error": "Usuario sin comunidad asignada"}, status=400)
    
    comunidad = user.comunidad
    # ... calculations ...
    
    return Response({
        "periodo": periodo_data,
        "kpis": { ... },
        "alertas": { ... },
        "charts": { ... }
    })
```

**Endpoint:** `GET /api/dashboard/kpis/`

<Note>
  All data is automatically scoped to the authenticated user's community. No additional filtering is needed on the frontend.
</Note>

## KPI Calculations

### Project Metrics

```python
# Total projects in community
total_proyectos = Proyecto.objects.filter(comunidad=comunidad).count()

# Active projects (in execution)
proyectos_activos = proyectos_qs.filter(estado='ejecucion').count()

# Closed projects
proyectos_cerrados = proyectos_qs.filter(estado='cierre').count()

# Total allocated budget across all projects
presupuesto_total = proyectos_qs.aggregate(
    Sum('presupuesto_total')
)['presupuesto_total__sum'] or 0
```

### Financial Metrics

```python
# Total paid amounts (actual expenditure)
total_pagado = Rendicion.objects.filter(
    proyecto__comunidad=comunidad, 
    estado='pagado'
).aggregate(Sum('monto_rendido'))['monto_rendido__sum'] or 0

# Total submitted renditions
total_rendido_ingresado = rendiciones_qs.aggregate(
    Sum('monto_rendido')
)['monto_rendido__sum'] or 0

# Total approved renditions
total_rendido_aprobado = rendiciones_qs.filter(
    estado='aprobado'
).aggregate(Sum('monto_rendido'))['monto_rendido__sum'] or 0

# Budget execution percentage
porcentaje_ejecucion_fin = (total_pagado / presupuesto_total * 100) if presupuesto_total > 0 else 0
```

```python
hoy = timezone.now().date()
proyectos_atrasados = proyectos_qs.filter(
    fecha_fin__lt=hoy
).exclude(estado__in=['cierre', 'rechazado']).count()
```

## Frontend Implementation

The dashboard uses React with Recharts for visualizations:

```jsx
import { PieChart, Pie, Cell, Tooltip, Legend, ResponsiveContainer } from 'recharts';

const COLORS = ['#6F574E', '#8a7871ff', '#ccb7afff', '#9CA3AF'];

export default function Dashboard() {
  const [data, setData] = useState(null);
  const [loading, setLoading] = useState(true);
  
  useEffect(() => {
    async function fetchDashboard() {
      const res = await apiFetch('/dashboard/kpis/');
      if (res.ok) {
        const json = await res.json();
        setData(json);
      }
      setLoading(false);
    }
    fetchDashboard();
  }, []);
  
  // ... render logic
}
```

## Utility Functions

### Currency Formatting

```javascript
const formatMonto = (amount) => 
  '$' + Number(amount).toLocaleString('es-CL', { maximumFractionDigits: 0 });

// Example: 45000000 → "$45.000.000"
```

### Date Formatting

```javascript
const formatFechaCL = (dateString) => {
  if (!dateString) return '...';
  const date = new Date(dateString + 'T00:00:00');
  return date.toLocaleDateString('es-CL', { 
    day: '2-digit', 
    month: '2-digit', 
    year: 'numeric' 
  });
};

// Example: "2024-12-31" → "31-12-2024"
```

## Real-time Updates

The dashboard fetches data on component mount. To enable real-time updates:

```jsx
useEffect(() => {
  const fetchDashboard = async () => {
    const res = await apiFetch('/dashboard/kpis/');
    if (res.ok) {
      const json = await res.json();
      setData(json);
    }
  };
  
  // Initial fetch
  fetchDashboard();
  
  // Poll every 30 seconds
  const interval = setInterval(fetchDashboard, 30000);
  
  return () => clearInterval(interval);
}, []);
```

<Warning>
  Frequent polling can increase server load. Consider using WebSockets or Server-Sent Events for true real-time updates in production.
</Warning>

```python
        return Response(cached_data)

    # Cache for 5 minutes

```
