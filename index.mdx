## Technology Stack

Auditoriapp is built with modern, production-ready technologies:

- **Backend**: Django 5.0 + Django REST Framework
- **Authentication**: JWT with refresh tokens (SimpleJWT)
- **Database**: PostgreSQL/MySQL with Django ORM
- **Frontend**: React 19 + Vite + TailwindCSS
- **Visualizations**: Recharts for interactive dashboards
- **Documentation**: OpenAPI/Swagger with drf-spectacular
