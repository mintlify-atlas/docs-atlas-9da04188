---
title: 'Communities API'
description: 'API endpoints for managing communities, councils, and members'
---

The Communities API allows you to manage communities (comunidades), councils (consejos), and members (socios) within the Auditoriapp platform.

## Base URL

All communities endpoints are prefixed with:

```
/api/comunidades/
```

## Authentication

All endpoints require JWT authentication. Include the token in the Authorization header:

```
Authorization: Bearer <your_jwt_token>
```

---

## Communities

### List Communities

<CodeGroup>
```bash cURL
curl -X GET "https://api.auditoriapp.com/api/comunidades/comunidades/" \
  -H "Authorization: Bearer YOUR_TOKEN"
```

```javascript JavaScript
const response = await fetch('https://api.auditoriapp.com/api/comunidades/comunidades/', {
  headers: {
    'Authorization': 'Bearer YOUR_TOKEN'
  }
});
const data = await response.json();
```

```python Python
import requests

response = requests.get(
    'https://api.auditoriapp.com/api/comunidades/comunidades/',
    headers={'Authorization': 'Bearer YOUR_TOKEN'}
)
data = response.json()
```
</CodeGroup>

<ResponseExample>
```json Response
[
  {
    "id": 1,
    "nombre": "Comunidad Los Andes",
    "activa": true,
    "creada_en": "2024-01-15T10:30:00Z",
    "usuario": 5
  }
]
```
</ResponseExample>

### Get Community Details

Retrieve detailed information about a specific community.

```
GET /api/comunidades/comunidades/{id}/
```

<ParamField path="id" type="integer" required>
  The unique identifier of the community
</ParamField>

<ResponseField name="id" type="integer">
  Unique identifier for the community
</ResponseField>

<ResponseField name="nombre" type="string">
  Name of the community
</ResponseField>

<ResponseField name="activa" type="boolean">
  Whether the community is active
</ResponseField>

<ResponseField name="creada_en" type="datetime">
  Timestamp when the community was created
</ResponseField>

<ResponseField name="usuario" type="integer">
  ID of the user associated with this community
</ResponseField>

### Create Community

Create a new community.

```
POST /api/comunidades/comunidades/
```

<ParamField body="nombre" type="string" required>
  Name of the community (max 255 characters)
</ParamField>

<ParamField body="activa" type="boolean" default={true}>
  Whether the community is active
</ParamField>

<ParamField body="usuario" type="integer">
  ID of the user to associate with this community
</ParamField>

<CodeGroup>
```bash cURL
curl -X POST "https://api.auditoriapp.com/api/comunidades/comunidades/" \
  -H "Authorization: Bearer YOUR_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{
    "nombre": "Nueva Comunidad",
    "activa": true
  }'
```

```javascript JavaScript
const response = await fetch('https://api.auditoriapp.com/api/comunidades/comunidades/', {
  method: 'POST',
  headers: {
    'Authorization': 'Bearer YOUR_TOKEN',
    'Content-Type': 'application/json'
  },
  body: JSON.stringify({
    nombre: 'Nueva Comunidad',
    activa: true
  })
});
```

```python Python
import requests

response = requests.post(
    'https://api.auditoriapp.com/api/comunidades/comunidades/',
    headers={'Authorization': 'Bearer YOUR_TOKEN'},
    json={
        'nombre': 'Nueva Comunidad',
        'activa': True
    }
)
```
</CodeGroup>

### Update Community

Update an existing community.

```
PUT /api/comunidades/comunidades/{id}/
PATCH /api/comunidades/comunidades/{id}/
```

<ParamField path="id" type="integer" required>
  The unique identifier of the community to update
</ParamField>

### Delete Community

Delete a community.

```
DELETE /api/comunidades/comunidades/{id}/
```

<ParamField path="id" type="integer" required>
  The unique identifier of the community to delete
</ParamField>

## Members (Socios)

### List Members

Retrieve members filtered by the authenticated user's community.

```
GET /api/comunidades/socios/
```

<Info>
Members are automatically filtered based on the authenticated user's community.
</Info>

<ResponseField name="id" type="integer">
  Unique identifier for the member
</ResponseField>

<ResponseField name="comunidad" type="integer">
  ID of the associated community
</ResponseField>

<ResponseField name="consejo" type="integer">
  ID of the associated council (nullable)
</ResponseField>

<ResponseField name="nombre" type="string">
  Name of the member (max 255 characters)
</ResponseField>

<ResponseField name="rol" type="string">
  Role of the member (max 50 characters)
</ResponseField>

<CodeGroup>
```bash cURL
curl -X GET "https://api.auditoriapp.com/api/comunidades/socios/" \
  -H "Authorization: Bearer YOUR_TOKEN"
```

```javascript JavaScript
const response = await fetch('https://api.auditoriapp.com/api/comunidades/socios/', {
  headers: {
    'Authorization': 'Bearer YOUR_TOKEN'
  }
});
```
</CodeGroup>

<ResponseExample>
```json Response
[
  {
    "id": 1,

    "consejo": 1,
```

## Permissions

- All endpoints require authentication
- Users can only access data from their associated community
- The `socios` endpoint automatically filters by the authenticated user's community
